# 独立进度看板

这个目录是和主项目界面隔离的独立进度看板。

它会直接读取并写回上一级的 `progress/` 真相源，并在保存成功后自动执行上一级的 `scripts/generate-progress.mjs`，同步更新：

- `项目开发进度.md`
- `当前执行看板.md`

## 启动方式

### 方式 1：直接启动

```powershell
Set-Location C:\Users\panshimao\IdeaProjects\layout3.0\progress-board
npm run start
```

然后打开：

`http://127.0.0.1:5070`

### 方式 2：一键打开

运行：

```powershell
.\open-board.ps1
```

脚本会尝试启动本地服务，并自动打开浏览器。

## 当前能力

- 任务看板：按状态列展示任务，可编辑任务状态、标题、优先级、验收标准与执行边界。
- 项目设置：可编辑阶段、里程碑、模块的状态与说明。
- 风险、技术债、日志：支持可视化修改与写回。
- 保存保护：保存前会比对真相源版本，避免把别人刚改过的内容静默覆盖掉。

## 当前不做

- 不嵌入现有 Electron 主应用。
- 不做远程同步、多用户协作、权限控制。
- 不做自动安装包与系统托盘常驻。
