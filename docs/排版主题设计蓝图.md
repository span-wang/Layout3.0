# 排版主题设计蓝图

## 一、设计理念

本设计系统围绕「简洁、清新、易读」三大原则，为不同使用场景定义四种排版主题。每种主题拥有独特的视觉调性，但共享统一的设计语言框架，确保用户在切换主题时获得连贯的体验。

### 核心原则

1. **无渐变**：所有视觉效果使用纯色填充，拒绝渐变色
2. **克制配色**：每种主题仅使用 3-5 种核心色彩，避免杂乱
3. **风格鲜明**：每种主题在字体、间距、装饰元素上有明确区分
4. **块类型一致**：所有主题支持相同的 11 种块类型渲染

## 二、块类型设计规范

### 2.1 块类型清单

| 块类型 | 英文标识 | 描述 | 典型场景 |
|--------|----------|------|----------|
| 标题 H1 | heading1 | 页面主标题 | 文档名称、章节标题 |
| 标题 H2 | heading2 | 章节标题 | 主要段落标题 |
| 标题 H3 | heading3 | 小节标题 | 子段落标题 |
| 段落 | paragraph | 正文内容 | 主要文字内容 |
| 列表 | list | 有序/无序列表 | 列举项、步骤说明 |
| 引用 | blockquote | 引用块 | 名言、参考资料 |
| 代码 | code | 代码块 | 技术文档、示例代码 |
| 表格 | table | 数据表格 | 结构化数据展示 |
| 图片 | image | 图片及说明 | 插图、图表 |
| 公式 | equation | 数学公式 | 数学表达式 |
| 分隔线 | horizontalRule | 水平分隔 | 章节分隔 |

**调性关键词**：严谨、专业、留白舒适

| 属性 | 规范值 |

| 正文色 | #333333 |
| 背景色 | #FFFFFF |
| 边框色 | #E0E0E0 |
| 引用背景 | #F8F9FA |
| 代码背景 | #F5F5F5 |

| 级别 | 字号 | 字重 | 颜色 | 装饰 |
